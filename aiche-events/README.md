# AIChE NITR — Upcoming Events

Responsive animated event listing page for the AIChE NITR Web Development Induction Task.

## Features
- Fade-in on scroll (IntersectionObserver)
- Staggered card entrance animations
- 3D tilt effect on mouse move
- Image zoom on hover
- Gradient top border on cards
- Colored tag badges per category
- Sticky blurred header
- Category filter tabs
- Search bar (filters title, tag, description)
- Skeleton loading animation
- Dark / Light mode toggle
- Responsive grid (1 → 2 → 3 columns)

## Tech Stack
React · JavaScript · Vite · Tailwind CSS

## How to run

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel
1. Push to GitHub
2. Import at https://vercel.com → New Project
3. Click Deploy
